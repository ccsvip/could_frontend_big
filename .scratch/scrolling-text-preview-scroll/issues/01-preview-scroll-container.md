# 确定查看弹窗长列表滚动容器

Type: prototype
Status: resolved

## Question

如何在滚动文本查看弹窗中建立响应式的独立文本列表滚动容器，使标题和状态标签保持可见，文本列表在内容溢出时始终显示纵向滚动条，并在桌面和移动端都不超出视口？

## Answer

保留 Modal 标题与状态标签在独立滚动区域之外。文本卡片包裹在使用动态视口单位计算最大高度的容器中，设置 `overflow-y: scroll` 以始终占用并显示纵向滚动条；右侧内边距避免滚动条遮挡卡片内容。

Modal 宽度改用现有表单一致的 `min(92vw, 45rem)` 响应式约束。高度上限为视口高度扣除 Modal 标题、状态标签和安全边距后的剩余空间，窄屏同样遵循该限制。
